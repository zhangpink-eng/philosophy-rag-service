### /api/query
```json
{
  "query": "问题",
  "results": [
    {
      "text_zh": "...",
      "text_en": "...",
      "source": "文件名",
      "score": 0.xxxx,
      "dense_score": 0.xxxx,
      "sparse_score": 0.xxxx
    }
  ]
}
```

### /api/compare
```json
{
  "query": "问题",
  "before_rerank": [...],  // TOP10, RRF 排序
  "after_rerank": [...],   // TOP5, rerank 排序
  "llm_response": "生成内容..."
}
```

## 八、验证命令

```bash
# 1. Qdrant 状态
curl http://localhost:6333/collections/philosophy_documents

# 2. API 服务状态
curl http://localhost:8000/

# 3. 检索测试
curl -X POST http://localhost:8000/api/query \
  -H "Content-Type: application/json" \
  -d '{"query": "什么是哲学实践"}'

# 4. 对比测试（无 LLM）
curl -X POST http://localhost:8000/api/compare \
  -H "Content-Type: application/json" \
  -d '{"query": "什么是哲学实践", "use_llm": false}'

# 5. 对比测试（带 LLM）
curl -X POST http://localhost:8000/api/compare \
  -H "Content-Type: application/json" \
  -d '{"query": "什么是哲学实践", "use_llm": true}'
```
